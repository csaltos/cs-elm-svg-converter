# CS Elm SVG Converter

Convert SVG files to Elm SVG code declarations

## Requirements

* JDK 1.8+
* Maven 3.6+

## Build

```bash
cd cs-elm-svg-converter
mvn package
```

## Run

```bash
cd cs-elm-svg-converter
java -cp target/cs-elm-svg-converter-*.jar < file.svg
```
